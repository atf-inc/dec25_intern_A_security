# Product Images

Place your product images in this folder with these exact filenames:

1. **iphone-15-pro.jpg** - iPhone 15 Pro image
2. **samsung-galaxy-s24.jpg** - Samsung Galaxy S24 image (cream/yellow one)
3. **macbook-pro-m3.jpg** - MacBook Pro M3 image
4. **dell-xps-15.jpg** - Dell XPS 15 image (colorful swirl wallpaper)
5. **airpods-pro.jpg** - AirPods Pro image
6. **sony-wh-1000xm5.jpg** - Sony WH-1000XM5 headphones image

## Image Requirements
- Format: JPG, PNG, or WebP
- Recommended size: 800x800px or larger
- Keep file sizes under 500KB for optimal loading

These images will be served from `/products/[filename]` in the Next.js application.
