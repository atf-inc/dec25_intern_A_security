# Configuration Guide

## Environment Variables

See `.env.example` for all available configuration options.

## Key Settings

- `CAPTURE_INTERFACE`: Network interface to monitor
- `DATABASE_URL`: PostgreSQL connection string
- `ML_MODELS_PATH`: Path to ML model files
- `SIGNATURE_ENGINE_ENABLED`: Enable/disable signature detection
- `AUTO_BLOCK_ENABLED`: Enable automatic IP blocking

## Configuration Files

- `config/settings.py`: Main configuration
- `config/tool_configs/`: Security tool configurations
- `config/policies/`: Security policies and rules
- `config/ml_configs/`: ML model parameters
